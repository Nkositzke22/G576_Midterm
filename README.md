# Bird Sighting Tracker

### Description
The Bird Sighting Tracker is an interactive web mapping application developed as my midterm project for G576 at UW Madison. The application allows users to view, add, and manage bird sightings within the Madison, Wisconsin area utilizing the ArcGIS Maps SDK for JavaScript.

### Features
- **Interactive Map**: A hybrid satellite basemap view centered on the Madison, Wisconsin region.
- **Search Functionality**: Integrated search widget to quickly find locations.
- **Geolocate**: A dedicated button to zoom the map to the user's current location.
- **Data Editing**: A full CRUD (Create, Read, Update, Delete) interface allowing users to contribute bird sighting data directly to the map via an integrated editor widget.

### Technologies Used
- **ArcGIS Maps SDK for JavaScript**
- **HTML5 & CSS3**
- **JavaScript (ES6+)**

### How to Use
You can view the live application directly in your web browser via GitHub Pages
